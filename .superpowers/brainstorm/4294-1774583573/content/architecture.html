✅ 이 설계로 진행할까요?

NEIS API (학사일정 + 급식) + 즉시 적용 개선사항을 1차로 구현하고, 이후 중간/고급 기능을 순차적으로 추가합니다. 모든 기능은 기존 Vanilla JS + Supabase 스택을 유지합니다.
