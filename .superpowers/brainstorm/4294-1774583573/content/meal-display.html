급식 표시 방식 선택

시간표 옆에 급식 버튼을 어떻게 배치할까요?

A

시간대별 인라인 버튼

아침(7:30) / 점심(12:30) / 저녁(18:30) 시간 행 옆에 "🍚 급식보기" 작은 버튼을 인라인으로 배치. 클릭 시 모달로 메뉴 표시

07:30 아침
1교시 — 수학(김선)
12:30 점심
18:30 저녁
B

사이드패널 상단 급식 섹션

오른쪽 사이드패널 맨 위에 "오늘의 급식" 접기/펼치기 섹션 추가. 아침/점심/저녁 탭으로 전환. 시간표와 별도 영역으로 분리

🍽️ 오늘의 급식 ▼
흑미밥 / 미역국 / 제육볶음
시금치나물 / 배추김치 / 요거트
C

A + B 조합

시간표에 인라인 버튼도 있고, 사이드패널 상단에도 급식 섹션 표시. 인라인 버튼 클릭 시 해당 끼니 탭으로 스크롤. 가장 접근성 좋음
